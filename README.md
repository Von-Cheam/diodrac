# diodrac

### Installation

```ShellSession
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install git

zef update
zef upgrade
zef install Terminal::ANSIColor

mkdir -p ~/Perl6_Code
cd ~/Perl6_Code
git clone https://github.com/Von-Cheam/diodrac.git
```
### Usage
```ShellSession
cd ~/Perl6_Code/diodrac
perl6 diodrac.pl6
```

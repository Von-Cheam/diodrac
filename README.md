### Installation

```shell
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install git

zef update
zef upgrade
zef install Terminal::ANSIColor

mkdir -p ~/Perl6_Code
cd ~/Perl6_Code
git clone https://github.com/Von-Cheam/dioidrac.git
```
### Usage
```shell
cd ~/Perl6_Code/dioidrac

perl6 diodrac.pl6
```
